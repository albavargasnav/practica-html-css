## Indicación de archivos

```
· Práctica-html-css-main: Carpeta general donde se incluyen todos los archivos

· Index.html: Html de la práctica. Su Css correspondiente es style.css

· Proyectos-grid: Html de proyectos en grid. Sus css correspondientes son style.css y grid.css

· Registro-completado: Html de un supuesto registro completado cuando le das a registrar en el formulario

· Carpeta multimedia: Incluye todos los archivos multimedia de la práctica.
```
